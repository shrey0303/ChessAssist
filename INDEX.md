## üí° Key Concepts Explained

### Token
- **What:** Lichess API authorization key
- **Where to get:** https://lichess.org/account/oauth/token
- **More info:** [STREAMING_INTEGRATION.md#required-setup](./STREAMING_INTEGRATION.md)

### Streaming API
- **What:** Real-time WebSocket-like connection to Lichess
- **Benefits:** Instant move detection, lower latency
- **More info:** [IMPLEMENTATION_SUMMARY.md#streaming-integration-details](./IMPLEMENTATION_SUMMARY.md)

### Rate Limiting
- **What:** Prevents too many API calls
- **How:** 800ms minimum between analyses per game
- **More info:** [STREAMING_INTEGRATION.md#rate-limiting-details](./STREAMING_INTEGRATION.md)

### Message API
- **What:** Communication between background service worker and popup
- **Types:** 7 event types (gameFull, analysis, etc.)
- **More info:** [STREAMING_INTEGRATION.md#message-api](./STREAMING_INTEGRATION.md)

## üêõ Troubleshooting Guide

| Problem | First Check | Solution |
|---------|-------------|----------|
| "No token found" | [INSTALLATION.md](./INSTALLATION.md#step-4-configure-token) | Paste token in popup |
| "Unauthorized" | [INSTALLATION.md](./INSTALLATION.md#troubleshooting-during-first-run) | Check token is valid |
| "Analysis not showing" | [INSTALLATION.md](./INSTALLATION.md#troubleshooting-during-first-run) | Wait 2+ seconds |
| "Game not detected" | [STREAMING_INTEGRATION.md](./STREAMING_INTEGRATION.md#troubleshooting) | Make sure game is on play.lichess.org |
| "Stream disconnects" | [STREAMING_INTEGRATION.md](./STREAMING_INTEGRATION.md#troubleshooting) | Auto-reconnect should handle it |
